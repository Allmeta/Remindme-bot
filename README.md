## To run you need...
* `echo {} > jobs.json`
  * // Should probs automate creation of the file
* `echo {"token"="<TOKEN>"} > config.json`
  * <TOKEN> is your secret discord bot token
* `pip3 install discord.py apscheduler`
## Then finally
`python3 bot.py`
